    baseline

    A historical view of pings sent for the org-mozilla-vrbrowser/baseline document type.

    This view is guaranteed to contain only complete days (per submission_timestamp) and to contain only one row per distinct document_id within a given date.

    Clustering fields: normalized_channel, sample_id

    • Friendly name: Historical Pings for org-mozilla-vrbrowser/baseline
    Referenced Tables
    Project Dataset Table
    moz-fx-data-shared-prod org_mozilla_vrbrowser_stable baseline_v1
